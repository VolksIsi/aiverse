## Manual Steps to Enable GitHub Pages

1. **Open Repository**: https://github.com/VolksIsi/aiverse
2. **Go to Settings**: Click the "Settings" tab
3. **Find Pages**: In the left sidebar, click "Pages"
4. **Configure Source**:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
   - Click "Save"
5. **Wait**: GitHub Pages will take 1-2 minutes to deploy
